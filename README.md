## Formulaire de contact

Le formulaire utilise [Formspree](https://formspree.io). Pour l'activer :

1. Créer un compte gratuit sur [formspree.io](https://formspree.io)
2. Créer un nouveau formulaire et copier votre ID
3. Dans `contact.html`, remplacer `YOUR_FORM_ID` par votre ID dans la ligne :
   ```html
   action="https://formspree.io/f/YOUR_FORM_ID"
   ```

## Déploiement sur GitHub Pages

1. Créer un repository GitHub (ex: `portfolio`)
2. Pousser tous les fichiers sur la branche `main`
3. Aller dans **Settings → Pages**
4. Choisir la branche `main` comme source
5. Le site sera accessible à `https://votre-username.github.io/portfolio`
